## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/alifiansya/portfolio-astro.git
    cd portfolio-astro
    ```

2. **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3. **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4. **Open your browser:**  
   Visit `http://localhost:4321` (or the port shown in your terminal).

### Build for Production

```bash
npm run build
# or
yarn build
```
The static site will be generated in the `dist/` directory.

## Customization

- Edit content in `src/content` or the relevant `.astro`/`.md` files.
- Update styles in `src/styles/`.
- Change theme or layout components in `src/components/`.

## Deployment

You can deploy the generated `dist/` folder to any static hosting, such as Vercel, Netlify, GitHub Pages, or your own server.
